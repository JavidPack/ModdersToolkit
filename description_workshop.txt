ModdersToolkit is a toolkit for modders, hence the name.

Current Tools
-C# REPL, evaluate and run code interactively (on server console as well)
-Hitbox visualizer
-Dust tool
-Item defaults tool
-Projectile spawner
-Player layer viewer
-Interface layer viewer.
-NPC Spawn Calculator
-Fishing Result Calculator (Thanks direwolf420)
-Texture updating/editing tool
-Shader updating/editing tool (w/ Webmilio) (see wiki for usage instructions)
-UI Playground
-Background Tool
-Miscellaneous (NPC info, tile grid, chunk calculator)

Planned Tools
-WorldGen tools
-Sound explorer/tester
-NPC Loot validator
-NPC Spawn validator
-NPC kill visualizer
